================================================================================
                      GESTURE DETECTION ALGORITHMS
================================================================================

Helper Functions Needed:

1. calculateDistance(point1, point2):
   - Euclidean distance between two landmarks
   - return Math.sqrt((x2-x1)² + (y2-y1)²)

2. calculateHandSpread(hand):
   - Average distance from palm center to all fingertips
   - return average of [distance(palm, thumb), distance(palm, index), ...]

3. isHandOpen(hand):
   - return calculateHandSpread(hand) > threshold (100px)

4. calculateHandRotation(hand, previousHand):
   - Compare palm normal vector between frames
   - return angle in degrees

5. detectPinch(hand):
   - Check if any finger tip distance < 20px from thumb tip
   - return pinched_finger_index or null

6. calculateHandVelocity(hand, previousHand):
   - Compare palm position between frames
   - return pixels_per_frame

7. calculateInterHandDistance(hand1, hand2):
   - Distance between hand centers
   - return distance

8. drawString(x1, y1, x2, y2, color, thickness, tension):
   - Draw line with glow, gradient, and effects
   - Handle all visual properties

GESTURE DETECTION LOGIC:

function detectCurrentGesture(hands, previousHands) {
  
  if (hands.length === 0) return null;
  if (hands.length === 1) return "SINGLE_HAND";
  
  const hand1 = hands[0];
  const hand2 = hands[1];
  
  const spread1 = calculateHandSpread(hand1);
  const spread2 = calculateHandSpread(hand2);
  const interDistance = calculateInterHandDistance(hand1, hand2);
  const velocity1 = calculateHandVelocity(hand1, previousHands[0]);
  const velocity2 = calculateHandVelocity(hand2, previousHands[1]);
  
  // HANDS APART check
  if (spread1 > 90 && spread2 > 90 && interDistance > 200) {
    return "HANDS_APART"; // CONSTELLATION
  }
  
  // HANDS CLOSE check
  if (interDistance < 150) {
    return "HANDS_CLOSE"; // MANDALA
  }
  
  // ROTATING check
  const rot1 = calculateHandRotation(hand1, previousHands[0]);
  const rot2 = calculateHandRotation(hand2, previousHands[1]);
  if (Math.abs(rot1) > 40 || Math.abs(rot2) > 40) {
    return "ROTATING"; // SPIRAL
  }
  
  // WEAVING check (hands moving opposite directions while close)
  if (interDistance < 200 && velocity1 > 300 && velocity2 > 300) {
    if (velocity1.direction opposite velocity2.direction) {
      return "WEAVING"; // TAPESTRY
    }
  }
  
  // PINCH check
  const pinch1 = detectPinch(hand1);
  const pinch2 = detectPinch(hand2);
  if (pinch1 !== null || pinch2 !== null) {
    return "PINCH"; // SACRED GEOMETRY
  }
  
  // DEFAULT
  return "IDLE";
}

================================================================================
                          PARTICLE SYSTEM DETAILS
================================================================================

CLASS: Particle

Properties:
- x, y: Position
- vx, vy: Velocity
- size: Radius in pixels
- color: RGBA color string
- lifetime: Remaining frames (0-255)
- maxLifetime: Total frames to display
- alpha: Current opacity (0-1)
- glowSize: Blur radius for glow effect
- theme: Current theme color

Methods:
- update(): Update position based on velocity, apply gravity/buoyancy
- draw(ctx): Render particle with glow effect
- isAlive(): Return true if lifetime > 0

Emission Rules:

onGestureChange():
  - Emit 30-50 particles from hand center
  - Scatter velocity in all directions
  - Duration: 200ms burst
  - Color: Accent color from theme
  - Size: 2-4px

onHandMovement():
  - If hand velocity > 300px/s:
    * Emit 5-8 particles per frame
    * From fingertip positions
    * Direction: Opposite to hand movement
    * Color: Primary theme color
    * Size: 1-2px

onStringCreation():
  - For every 50px of string length:
    * Emit 1 particle
    * Position: Randomly along string
    * Velocity: Gently drift upward
    * Color: Primary or secondary (alternate)
    * Size: 0.5-1px

ParticlePool Management:
- Maintain pool of ~500 pre-allocated particles
- Reuse particles instead of creating/destroying
- Cap active particles at 2000 maximum
- Remove oldest particles when at capacity
